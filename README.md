# SimAgentCity v1.0

## The First Truth (Seed Axiom)
"Physical UI-to-OS Synchronization: Moving a pixel on a grid executes a file-system operation on the host."

## Overview
SimAgentCity is a conceptual orchestrator that visualizes your local file system and AI agent factory as a tile-based simulation grid inspired by SimCity 1995. 

### Features
- **OS Bridge:** Real-time synchronization between UI entity coordinates and file system paths.
- **Registry Plaza:** 2D visualization of Windows Registry keys as interactable buildings.
- **Agent Mall:** Recruitment center where you can download or register new agent sims into the population.
- **AI-Compatible Tools:** Includes `task_mgr_mini` for lightweight process monitoring and `agent_registrar_tool` for population hooks.
- **Genetic Build Engine:** All modules are generated and verified through an AST Veto and Natural Selection process.
- **Concurrent Standby:** Multiple h2o-danube instances are ready to process files dragged into 'Industrial' zones.
- **Retro Aesthetic:** Lightweight Vanilla JS/CSS rendering with a 1.2s Pulse Handshake.

## Districts
1. **FILE MALL:** Browse and manage your `city_workspace` files.
2. **REGISTRY PLAZA:** Visualizes `HKCU\Software` keys as system buildings.
3. **AGENT DISTRICT:** Home to your recruited Sims (Miners, Processors, Shippers).
